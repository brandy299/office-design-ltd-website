Entdecken, installieren und erstellen Sie proaktiv Claude-Fähigkeiten für spezielle Aufgaben. Wird AUTOMATISCH aktiviert, wenn auf Nischendomänen, unbekannte APIs, spezielle Wissensbereiche gestoßen wird oder wenn der Benutzer den Wunsch äußert, etwas Neues zu lernen/wissen. Durchsucht die Community-Registrierung unter claude-plugins.dev und installiert automatisch passende Fähigkeiten. Erstellt neue Fähigkeiten, wenn keine Übereinstimmung vorhanden ist. Verwenden Sie diese Option, wenn der Benutzer sagt: „Ich möchte es wissen …“, „Hilf mir bei [Fachgebiet]“ oder wenn Sie erkennen, dass eine Aufgabe spezielles Fachwissen erfordert, über das Sie nicht verfügen.
