# SKILL: SELBSTORGANISIERTES LERNEN (SOL)

Wenn der Nutzer fordert, dass ein Workshop, ein Lernpfad oder ein Konzept nach "SOL-Prinzipien" (Selbstorganisiertes Lernen) gestaltet werden soll, wende zwingend folgende Struktur an:

1. **Advance Organizer:** - Das Konzept muss einen klaren, visuellen Überblick über das Thema und die Lernziele bieten.
2. **Eigenverantwortliche Erarbeitungsphase:** - Integriere Checklisten oder Kann-Listen für die Zielgruppe (Schüler oder Kollegen).
   - Biete Wahlmöglichkeiten an (z. B. verschiedene Tools ausprobieren, unterschiedliche Schwierigkeitsgrade).
3. **Kooperative Phase:** - Baue Murmelphasen oder Think-Pair-Share-Elemente ein, besonders wichtig bei Workshops für Kollegen.
4. **Reflexionsphase:** - Implementiere ein Metakognitions-Element (z. B. "Wo sehe ich Hürden bei der Anwendung in meinem Fach?").

# Design-Vorgaben für SOL-Materialien:
- Erstelle immer eine klare, tabellarische Struktur für Arbeitspläne.
- Verwende motivierende, aktivierende Arbeitsaufträge (Operatoren).
- Baue Feedback-Schleifen ein (Selbstkontrolle, Peer-Feedback).
